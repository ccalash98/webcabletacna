Terminos y Condiciones

1) El contrato entrará en vigencia a partir de su instalación y/o activación efectiva, por un periodo forzoso de 6 meses, luego del cual el periodo pasará a ser de plazo indeterminado.


2) Si el contrato es resuelto por EL CLIENTE o por causas imputables a este durante el periodo forzoso (06 meses), EL CLIENTE pagará como penalidad el equivalente de 100.00 soles.


3) Si el contrato es modificado a solicitud de EL CLIENTE por causas imputables a este durante el periodo forzoso de (06 meses) para cambio de PLAN, si aumenta se exonera penalidad, si disminuye EL CLIENTE pagará como penalidad el equivalente de 45.00 soles.

5) En caso de corte temporal se extenderá el tiempo de permanencia forzoso, agregando el tiempo que el servicio estuvo suspendido.
